# Bitaxe Monitor - Windows Distribution

## Installation

### Option 1: Using the Installer (Recommended)
1. Download `BitaxeMonitorInstaller.exe`
2. Run the installer as Administrator
3. Follow the installation wizard
4. Launch from Start Menu or Desktop shortcut

### Option 2: Portable Executable
1. Download `BitaxeMonitor.exe`
2. Create a folder for the application (e.g., `C:\BitaxeMonitor`)
3. Copy the executable to this folder
4. Run `BitaxeMonitor.exe`

## System Requirements
- Windows 10 or Windows 11
- .NET Framework 4.7.2 or later (usually pre-installed)
- Network connection to access your Bitaxe device

## First Run Setup
1. Launch the application
2. Go to the Settings tab
3. Enter your Bitaxe device IP address
4. Configure your mining pool settings
5. Optionally set up email notifications

## Features
- Real-time monitoring of Bitaxe Gamma 601 mining devices
- Historical performance charts and analytics
- Automated optimization algorithms
- Email notifications for alerts
- Performance data logging and analysis

## Troubleshooting

### Application Won't Start
- Ensure you have Windows 10/11
- Try running as Administrator
- Check Windows Defender/Antivirus isn't blocking it

### Connection Issues
- Verify your Bitaxe device IP address in Settings
- Ensure both devices are on the same network
- Check that the Bitaxe web interface is accessible

### Performance Issues
- Close other resource-intensive applications
- Ensure adequate system RAM (4GB+ recommended)

## Support
For issues and questions, please check the application logs in the installation directory.
